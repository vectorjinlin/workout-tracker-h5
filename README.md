# 运动基数 Tracker

这是一个基于 React + Vite + TypeScript + TailwindCSS + Framer Motion + Recharts 的移动优先 H5 运动记录页面。

## 启动步骤

1. 安装依赖

```bash
npm install
```

或者使用你本机可用的包管理器：

```bash
yarn install
```

2. 启动开发服务器

```bash
npm run dev
```

3. 在浏览器中打开本地地址，例如 `http://localhost:5173`

## 目录结构

- `index.html` - React 根页面
- `src/App.tsx` - 主 UI 和交互
- `src/main.tsx` - 入口文件
- `src/index.css` - Tailwind 样式
- `tailwind.config.js` / `postcss.config.js` - 样式配置
